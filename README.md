# Malicious URL Detection using Deep Learning

## Description
This project classifies URLs as Phishing or Legitimate using an MLP model.

## Dataset
UCI Phishing Websites Dataset

## Model
- MLP (Dense Layers)
- Dropout, Early Stopping
- Adam Optimizer

## Results
Accuracy: ~95%
